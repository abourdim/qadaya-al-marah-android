# 🌹 Women's Issues — قضايا المرأة

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **قضايا المرأة بين التقاليد الراكدة والوافدة** (Women's Issues Between Stagnant and Imported Traditions) by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 issues on women's rights in Islam with truth vs misconception format
- Heroines section: women who shaped Islamic history
- Daily habit tracker with streaks
- Knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Narrator with karaoke, sleep timer, and lock screen controls
- Responsive design + print styles
- PWA-ready with manifest.json

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort by a Muslim parent to share Sheikh al-Ghazali's wisdom on women's rights interactively. Content derived from the book and trusted Islamic sources. This is not a fatwa.
